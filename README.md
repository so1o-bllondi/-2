**Визуализация графа зависимостей**  
Описание  
Инструмент для анализа и визуализации графов зависимости пакетов. Выполнен с 
поддержкой NuGet пакетов и тестовых репозиториев. Реализован в 5 этапов 
функциональности.  
**Этап 1:**
Минимальный прототип с конфигурацией 
Создано минимальное CLI-приложение, опции командной строки – источники 
настраиваемых параметров 
Аргументы командной строки: --package – имя анализируемого пакета, --url – URL 
репозитория пакетов, --test-repo – путь к файлу тестового репозитория, --test-mode – режим 
работы с тестовым репозиторием, --reverse – режим обратных зависимостей для пакета, -
visualize – режим визуализации графа, --output – имя выходного файла 
Графический интерфейс - окно с областью вывода и строкой ввода  
**Этап 2:**
Сбор данных 
Реализована основная логика получения данных о зависимостях для их дальнейшего анализа 
и визуализации с помощью формата пакетов .NET (NuGet).   
**Этап 3:**
Построение графа зависимостей  
Получение графа зависимостей, реализация алгоритма DFS 
Обработан случай циклических зависимостей   
**Этап 4:**
Дополнительные операции 
Поиск и поддержка вывода на экран обратных зависимостей пакета –reverse   
**Этап 5:**
Визуализация графа зависимостей 
Формирование текстового представления графа зависимостей на языке диаграмм D2 
Сохранение изображения графа в различных форматах
